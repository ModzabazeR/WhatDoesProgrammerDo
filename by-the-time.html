กว่าจะมาเป็นเว็บนี้

เว็บนี้สร้างขึ้นโดยไม่อาศัยโปรแกรมสำเร็จรูป

The Making Of : What Does Programmer Do

เนื่องจากเว็บนี้สร้างขึ้นโดยไม่ใช้โปรแกรมสำเร็จรูป แต่สร้างขึ้นโดยการเขียนโค้ด HTML, CSS และ JS ไปเรื่อย ๆ เท่านั้น ดังนั้นจึงต้องใช้เวลามาก และเจอปัญหาอุปสรรคไม่น้อย เราจะพาคุณไปพบกับขั้นตอนการสร้างเว็บนี้ตั้งแต่ต้นจนจบ


ระหว่างการทำงานของผม เครื่องมือที่ผมใช้นั้นมีไม่กี่อย่าง ก็คือโปรแกรม Visual Studio Code และWeb Hosting สิ่งที่ผมต้องทำก็คือ เขียนโค้ด HTML CSS และ JavaScript เอง ไปดูตัวอย่างคนอื่นมาบ้าง copy มาบ้าง และทำเองบ้าง ทำให้ผมได้อะไรมากมายจากการสร้างเว็บนี้ขึ้นมา ทั้งทักษะที่มากขึ้น ทั้งวิชาชีพ และวิชาชีวิตครับ คลิปวีดิโอด้านล่างนี้ จะเป็นคลิปเบื้องหลังการทำงานของผมตั้งแต่ต้นจนถึงตอนนี้ ผมอยากให้ทุก ๆ คนได้รับชมนะครับ ขอบคุณครับ
